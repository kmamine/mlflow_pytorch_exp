# mlflow_pytorch_exp    


1. Build  the packege:

```bash
python setup.py sdist
```

2. Install package :

```bash
pip install dist/mlflow-pytorch-exp-0.1.0.tar.gz
```

3. Create project 
```bash
mlflow-pytorch-exp create <path/to/folder>```
```

example : 
```bash
mlflow-pytorch-exp create my_project
```
